Run a query from command line :
```
psql -d mydatabase -c "SELECT * FROM my_table"
```

Créer un user :
```
CREATE USER rails WITH PASSWORD ‘rails’;
```

Voir les users :
```
\du
```

Lister les bases:
```
\l
```

Se connecter à une base en particulier
```
\c mrdrive-dev
```

Lister les tables de la base courante:
```
\dt
```
